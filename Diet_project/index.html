Growth

a linux based command line application for body growth

In order to build muscle or to have a perfect diet for living a beautiful life an application is needed which keeps the backup of all necessary activities like taking food, doing workouts on particular timings to burn calories. GROWTH application will take care of all your needs. This program has an authorized feature so one person's data is not visible to another person. It is a command line interface application even the user interface is quite simple to understand and operate. When a user enters data into an application everything is stored in a particular file so it is easy access from internal storage. C++ language is used as a base language in this application so execution is very fast and secure (because of OOP). MYSQL database is used for storing each user's details.

STARTING MARIADB MYSQL SERVER

Using service mariadb start command database server is started

RUNNING THE APPLICATION

Here you have to link the library "mysql" so check the respective path in /usr/include/ directory of your system

clang++ main.cpp -L/../../usr/include/mariadb/mysql -lmysqlclient -o a && ./a

After running the application this is the first look.
